# ComfyUI-VdoFlash

A robust ComfyUI Custom Node for orchestrating fully automated image-to-video pipelines with dynamic fallbacks, memory management, and intelligent model validation handling.

## Features
- **Validation Engine:** Built-in VRAM constraint checks and dimension auto-correction.
- **Fail-Safe Video Orchestration:** Will intelligently fallback to image frame-repetition if video graphs fail to compile or OOM.
- **VRAM Aware Continuity Mode (WIP)** 

## Installation

```bash
cd ComfyUI/custom_nodes/
git clone <repository_url>
```
